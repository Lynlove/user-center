# 项目简介
## 项目介绍
帮助大家找到学习伙伴的移动端 H5 网站(APP 风格)，
基于 Spring Boot 后端 + Vue3 前端的 全栈项目，包括
用户登录、更新个人信息、按标签搜索用户、建房组队、推荐相似用户等功能。
